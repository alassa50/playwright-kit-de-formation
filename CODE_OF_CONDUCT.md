# Code de conduite

Ce projet adopte le **Contributor Covenant v2.1**.

## Notre engagement

Nous nous engageons à maintenir un espace ouvert, inclusif et respectueux.

## Comportements attendus

- Respect et bienveillance.
- Feedback constructif.
- Collaboration orientée apprentissage.

## Signalement

Contactez les mainteneurs du dépôt via une issue privée ou un canal défini par l'organisation.

Texte de référence complet : https://www.contributor-covenant.org/version/2/1/code_of_conduct/
