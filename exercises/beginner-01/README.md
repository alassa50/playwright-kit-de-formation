# Exercice Débutant 01 — Login basique

## Énoncé

Automatiser la connexion d'un utilisateur sur une page de démonstration.

## Critères d'acceptation

1. Le formulaire accepte email et mot de passe.
2. Le message de succès apparaît après soumission.
3. Le test utilise des sélecteurs accessibles (`getByLabel`, `getByRole`).

## Fichiers

- `tests/login.spec.ts`
- `starter/login-page.ts`
