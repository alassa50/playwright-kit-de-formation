# Exercice Débutant 00 — Bases JavaScript/TypeScript

## Énoncé

Lire, exécuter et modifier une application de panier d'achat pour pratiquer les
constructions essentielles de JavaScript/TypeScript dans un contexte Playwright.

## Critères d'acceptation

1. Comprendre l'interface `Produit` définie dans `starter/cart-helpers.ts` et identifier
   les quatre champs qu'elle expose.
2. Lire `calculerTotal` et expliquer pourquoi `filter` est appelé avant `reduce`.
3. Lire `formaterLigne` et identifier les trois concepts JS/TS utilisés (template
   literal, destructuring, opérateur ternaire).
4. Exécuter les tests et vérifier qu'ils sont tous au vert.
5. **Bonus** : ajouter un produit à la liste `produits` dans le fichier de test et
   vérifier que les assertions restent correctes.

## Fichiers

- `tests/cart.spec.ts`
- `starter/cart-page.ts`
- `starter/cart-helpers.ts`
